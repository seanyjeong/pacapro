# P-ACA v1.3.3 Changelog

**릴리즈 날짜:** 2025-12-03

## 새로운 기능

### 체험생(Trial Student) 기능
- 체험생 등록 기능 추가 (2회 무료 체험)
- 체험생 전용 탭 추가 (학생 관리 페이지)
- 체험생 수정 기능 (체험 일정 변경 가능)
- 체험생 정식 등록 전환 기능
- 스케줄 캘린더에 체험생 수 표시 (보라색 ✨ 아이콘)
- 타임슬롯 모달에 체험생 배지 표시 (체험 1/2, 2/2)

### UI/UX 개선
- 스케줄 캘린더 넓이 확대 (2/3 → 3/4)
- 강사 배정 패널 넓이 축소 (1/3 → 1/4)
- 미배정 출근 요청 모달 패딩 개선
- 학생 목록에 체험생 배지 표시

## 버그 수정

### 체험생 관련
- 체험생이 재원생 목록에 포함되는 버그 수정
- 체험생 스케줄 배정 시 `class_schedule_id` 컬럼명 오류 수정
- `is_trial` 필터가 API에 전달되지 않는 버그 수정
- `is_trial`이 0일 때 "0"이 화면에 표시되는 버그 수정

### 알림톡 관련
- 미납 알림톡 SQL 오류 수정 (`year_month`, `final_amount` 컬럼)
- 납부기한을 학원 설정의 `tuition_due_day`로 표시하도록 수정

### 설정 관련
- `/settings/academy` API에 시간대 설정 누락 수정
