P-ACA 소개

P-ACA란?

P-ACA는 Physical Academy의 약자로, 체육대학 입시를 준비하는 학생들을 위한 학원관리시스템입니다. "Physical"은 체육·신체를, "Academy"는 학원을 의미하며, 체대입시 전문 학원의 체계적인 운영을 돕기 위해 만들어졌습니다.

로고의 의미

P-ACA 로고는 파란색 그라데이션 배경 위에 흰색 'P' 글자가 있고, 그 안에 역동적으로 달리는 사람의 형상이 담겨 있습니다.

이 디자인은 목표를 향해 전력 질주하는 체대입시 학생의 모습을 상징합니다. 파란색은 신뢰와 전문성을, 달리는 모습은 도전과 성취를 향한 열정을 나타냅니다.

주간 수업 월간 수업 (4주 기준)

1.2 5주차 수업

무료 수업 제공: 한 달이 5주인 경우, 5주차 수업은 추가 요금 없이 무료로 제공됩니다.

2. 학원비 규정

2.1 기본 계산

최종금액 = 기본요금 - 할인금액 + 추가금액
금액은 천원 단위로 절삭합니다. (예: 152,500원 → 152,000원)

2.2 납부 기한

  • 기본 납부일: 학원에서 설정한 날짜
  • 월중 등록 시: 등록일로부터 7일 이내

2.3 월중 등록 시 일할 계산

계산 방법:
일할요금 = 월요금 × (남은 수업일 ÷ 월 총 수업일)

예시: 주 2회 학생이 15일에 등록, 남은 수업일 3회
→ 월수강료 × (3÷8) = 약 37.5%

2.4 미납 처리

납부기한 초과 시 연체 상태로 표시되며, 설정에 따라 자동 알림 문자가 발송될 수 있습니다.

3.2 공결 규정

공결 인정 사유:

  • 질병
  • 학교 시험
공결 처리 방식:

학원은 학교가 아니므로, 공결은 출석으로 인정되지 않습니다.

① 5주차가 있는 달: 5주차 무료수업으로 공결분을 대체합니다. (별도 정산 없음)

② 5주차가 없는 달: 공결로 인해 월 수업 횟수가 기준(주 2회 = 8회)에 미달할 경우, 다음 달 학원비에서 해당 수업료만큼 차감됩니다.

예) 주 2회 학생이 공결 1회 발생 → 해당 월 7회 수업 → 다음 달 학원비에서 1회분 차감

3.3 보충 수업

결석 또는 공결 시, 가능하면 같은 달 다른 날짜에 보충수업을 받는 것을 권장합니다. 보충수업을 받으면 별도 정산이 필요하지 않습니다.

4. 휴원/복귀 규정

4.1 휴원 신청

필요 정보:

  • 휴식 시작일 (필수)
  • 휴식 종료일 (선택, 없으면 무기한)
  • 휴식 사유 (선택)

휴원 시 처리:

  • 수업 스케줄에서 제외
  • 학원비 생성 중단
  • 이월 크레딧 생성 (선택 시)

4.2 복귀 처리

복귀 시 남은 기간에 대해 일할계산으로 학원비가 자동 생성되며,
이월 크레딧이 있으면 자동으로 차감됩니다.

7. 체험 규정

7.1 체험 수업

  • 횟수: 최대 2회 무료 체험
  • 대상: 신규 상담 학생

7.2 정식 등록 전환

체험 완료 후 정식 등록 시 수업 요일과 시간대를 설정하고, 정규 학원비가 부과됩니다.

10. 개인정보보호 규정

10.1 개인정보보호법 준수

P-ACA 시스템은 개인정보보호법에 따라 민감한 개인정보를 암호화하여 저장합니다.

10.2 암호화 대상 정보

대상 암호화 항목
학생 정보이름, 전화번호, 학부모 전화번호, 주소
강사 정보이름, 전화번호, 주소, 주민번호, 계좌번호, 예금주
사용자 정보이름, 전화번호

10.3 보안 조치

  • 민감정보 암호화 저장 (AES-256 암호화)
  • 비밀번호 안전하게 암호화 저장
  • 로그인 인증 시스템
  • 사용자별 접근 권한 관리 (원장/관리자/직원 분리)
중요: 암호화 키 없이는 데이터베이스가 유출되더라도 개인정보를 확인할 수 없습니다.
