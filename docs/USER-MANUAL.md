## 1. 시작하기

### 1.1 접속하기

**컴퓨터에서 접속**
1. 인터넷 브라우저(크롬 권장)를 엽니다
2. 주소창에 `paca.vercel.app` 입력 후 엔터

**태블릿에서 접속**
1. 크롬 브라우저로 `paca.vercel.app/tablet` 접속
2. 메뉴 → "홈 화면에 추가" 선택
3. 앱처럼 바로가기가 생성됩니다

**휴대폰에서 접속**
1. 크롬 브라우저로 `paca.vercel.app/m` 접속
2. 메뉴 → "홈 화면에 추가" 선택
3. 앱 아이콘이 홈 화면에 생성됩니다

> **참고:** 휴대폰으로 PC 주소(`paca.vercel.app`)에 접속하면 자동으로 모바일 버전(`/m`)으로 이동됩니다.

### 1.2 회원가입

1. 로그인 화면에서 **"회원가입"** 클릭
2. 다음 정보를 입력합니다:

| 항목 | 설명 | 필수 |
|------|------|:----:|
| 학원명 | 학원 이름 | ✓ |
| 이메일 | 로그인 아이디로 사용 | ✓ |
| 비밀번호 | 영문+숫자 8자 이상 | ✓ |
| 대표자 이름 | 원장 이름 | ✓ |

3. **"가입하기"** 버튼 클릭
4. 가입 완료 후 로그인

### 1.3 로그인

1. 이메일 입력
2. 비밀번호 입력
3. **"로그인"** 버튼 클릭

**비밀번호를 잊으셨나요?**
1. 로그인 화면에서 **"비밀번호 찾기"** 클릭
2. 가입할 때 사용한 이메일 입력
3. 이메일로 전송된 링크를 클릭하여 새 비밀번호 설정

### 1.4 처음 설정 (온보딩)

처음 로그인하면 기본 설정 화면이 나옵니다:

1. **수업 시간 설정**
   - 오전반 시간 (예: 09:00~12:00)
   - 오후반 시간 (예: 14:00~17:00)
   - 저녁반 시간 (예: 18:00~21:00)

2. **학원비 설정**
   - 주 1회~7회 수업료 입력
   - 입시반/성인반 구분 설정

3. **강사 등록**
   - 기본 강사 정보 입력

4. **샘플 데이터 생성** (선택)
   - 체험용 샘플 데이터를 생성하여 기능 확인 가능

설정을 완료하면 대시보드로 이동합니다.

## 2. 대시보드 (홈 화면)

로그인 후 가장 먼저 보이는 화면입니다.

### 2.1 한눈에 보는 현황

화면 상단에 4개의 주요 통계 카드가 표시됩니다:

| 카드 | 설명 | 클릭 시 |
|------|------|---------|
| **수강 학생** | 현재 재원 중인 학생 수 | 학생 목록 이동 |
| **근무 강사** | 현재 근무 중인 강사 수 | 강사 목록 이동 |
| **이번 달 수입** | 이번 달 수납된 학원비 총액 (건수 포함) | 학원비 페이지 이동 |
| **순수익** | 이번 달 수입 - 지출 | 리포트 이동 |

### 2.2 오늘의 할 일

대시보드 중앙에 오늘 처리할 항목이 표시됩니다:

**미납 학원비**
- 아직 납부하지 않은 학원비 건수와 금액
- 클릭하면 학원비 페이지의 미납 필터로 이동

**휴원 종료 대기**
- 휴원 기간이 끝났지만 아직 복귀 처리되지 않은 학생
- 초과된 일수 표시
- 클릭하면 복귀 처리 모달 열림

**오늘 강사 일정**
- 오늘 각 시간대(오전/오후/저녁)에 근무할 강사 목록
- 클릭하면 수업 스케줄 페이지로 이동

**오늘 상담 일정**
- 오늘 예정된 상담 목록 (시간, 학생명, 상태)
- 클릭하면 상담 페이지로 이동

### 2.3 이번 달 현황

하단에 이번 달 재무 현황이 타임라인 형태로 표시됩니다:
- 수입 추이
- 지출 추이
- 순수익 변화

### 2.4 왼쪽 메뉴바

화면 왼쪽에 있는 사이드바를 통해 각 기능으로 이동합니다.

**사이드바 접기/펼치기:**
- 사이드바 상단의 접기 버튼을 클릭하면 아이콘만 표시
- 접힌 상태에서 아이콘에 마우스를 올리면 메뉴 팝업 표시
- 다시 클릭하면 펼쳐짐

**메뉴 구조:**

```
학원 운영
   ├── 학생
   ├── 강사
   ├── 수업스케줄
   ├── 수업일관리
   ├── 학원일정
   └── 시즌

재무 관리
   ├── 학원비
   ├── 급여
   ├── 지출
   ├── 수입
   └── 리포트

커뮤니케이션
   ├── 신규상담
   ├── 재원생상담
   ├── 문자 보내기
   └── 알림톡 설정

관리
   ├── 직원관리 (원장만)
   └── 설정
```

> **참고:** 직원 계정은 원장이 부여한 권한에 따라 보이는 메뉴가 다릅니다.

## 3. 학생 관리

왼쪽 메뉴에서 **"학생"** 클릭

### 3.1 학생 목록 보기

모든 학생이 표 형태로 표시됩니다.

**표시되는 정보:**
- 이름
- 연락처
- 학교/학년
- 수업 요일
- 월 수강료
- 상태 (재원/휴원/체험 등)

**7개 탭으로 분류:**

| 탭 | 설명 | 숫자 표시 |
|----|------|----------|
| 재원생 | 현재 수업 중인 학생 | ✓ |
| 휴원 | 일시적으로 쉬는 학생 | ✓ |
| 체험 | 체험 수업 중인 학생 | ✓ |
| 미등록관리 | 상담 후 아직 등록하지 않은 학생 | ✓ |
| 퇴원 | 학원을 그만둔 학생 | ✓ |
| 졸업 | 졸업한 학생 | ✓ |
| 학교별 | 학교별로 그룹화해서 보기 | - |

> v3.5.8부터 각 탭에 해당 학생 수가 숫자로 표시됩니다.

**학생 찾기:**
- 상단 검색창에 이름 또는 전화번호 입력
- 필터 옵션: 학년, 학생유형(입시생/성인), 입시전형(수시/정시), 성별

**엑셀 다운로드:**
- 학생 목록 상단의 **"엑셀 다운로드"** 버튼 클릭
- 학생명, 연락처, 학교, 학년 등 정보가 엑셀 파일로 저장됩니다

### 3.2 학생 등록하기

1. **"+ 학생 등록"** 버튼 클릭
2. 기본 정보 입력:

| 항목 | 설명 | 필수 |
|------|------|:----:|
| 이름 | 학생 이름 | ✓ |
| 학년 | 고1, 고2, 고3, N수, 중3 등 | ✓ |
| 성별 | 남/여 | ✓ |
| 학교 | 재학 중인 학교 | |
| 연락처 | 학생 전화번호 (010-0000-0000) | ✓ |
| 학부모 연락처 | 보호자 전화번호 | |
| 주소 | 거주지 주소 | |

3. 수업 정보 입력:

| 항목 | 설명 |
|------|------|
| 학생 유형 | 입시생 / 성인 |
| 수업 요일 | 월~일 중 선택 (여러 개 가능) |
| 시간대 | 오전 / 오후 / 저녁 |
| 주 수업 횟수 | 선택한 요일 수에 따라 자동 계산 |
| 월 수강료 | 설정의 요금표에서 자동 입력, 직접 수정 가능 |
| 할인 | 비율(%) 또는 금액 입력, 할인 사유 입력 |
| 납부 기한일 | 매월 납부일 (예: 5일) |

4. 추가 정보:

| 항목 | 설명 |
|------|------|
| 주소 | 학생 거주지 |
| 메모 | 특이사항 기록 |

> **참고:** 입시 전형, 목표 대학, 내신 평균, 모의고사 성적, 소개 학생 등의 학업 정보는 **상담 신청** 시 입력됩니다 (→ [19. 상담 신청 페이지](#19-상담-신청-페이지-공개-페이지) 참조).

5. **"저장"** 버튼 클릭

### 3.3 체험생 등록하기

체험 수업을 진행하는 학생을 등록합니다.

1. 학생 등록 화면에서 **"체험생으로 등록"** 체크
2. 체험 일정 입력:
   - 체험 날짜 선택 (여러 날 가능, 횟수 제한 없음)
   - 시간대 선택
3. **"저장"** 클릭

> v3.6.4부터 체험 일정 횟수 제한이 해제되어 여러 번의 체험을 자유롭게 설정할 수 있습니다.

**체험 후 처리:**
- 정식 등록: 학생 상세 → **"정식 등록"** 버튼 → 수업 정보 입력 → 저장
- 미등록 관리: 상태를 "미등록"으로 변경하여 추후 관리
- 퇴원 처리: 체험 후 등록하지 않는 경우

### 3.4 학생 정보 수정하기

1. 학생 목록에서 수정할 학생 클릭
2. 학생 상세 화면에서 **"수정"** 버튼 클릭
3. 정보 수정 후 **"저장"**

### 3.5 학생 상세 정보

학생을 클릭하면 상세 정보를 볼 수 있습니다:

**기본 정보 탭**
- 연락처, 학교, 학년, 주소 등
- QR 코드 (출석 체크용)

**학원비 탭**
- 납부 내역 (월별)
- 미납 현황
- 크레딧 잔액
- 선납 결제 내역

**출석 탭**
- 월별 출석 현황
- 출석/결석/지각/공결 통계

**상담 탭**
- 상담 기록 이력
- 목표 대학 정보
- 입시 전략 메모

**시즌 탭**
- 시즌 등록 이력
- 시즌비 납부 현황
- 환불 이력

### 3.6 휴원 처리하기

학생이 일정 기간 쉬어야 할 때 사용합니다.

1. 학생 상세 화면에서 **"휴원"** 버튼 클릭
2. 휴원 처리 모달이 열립니다:
   - 휴원 시작일 입력
   - 휴원 종료일 입력
   - 휴원 사유 입력 (선택)
   - 크레딧 옵션 선택: 이월 / 환불 / 없음
3. **"휴원 처리"** 클릭

**휴원 시 자동 처리:**
- 이번 달 학원비가 일할 계산으로 자동 조정됩니다
- 수업 스케줄에서 해당 학생이 제외됩니다
- 환불 선택 시 자동으로 환불 기록이 생성됩니다 (v3.6.0)

**휴원 복귀:**
- 종료일이 되면 대시보드에 "휴원 종료 대기" 알림이 표시됩니다
- 해당 알림을 클릭하면 복귀 처리 모달이 열립니다
- **"복귀"** 버튼 클릭 → 수업 스케줄에 자동 재배정

### 3.7 퇴원 처리하기

1. 학생 상세 화면에서 **"퇴원"** 버튼 클릭
2. 퇴원일 선택
3. 퇴원 사유 입력 (선택)
4. **"퇴원 처리"** 클릭

**퇴원 시 자동 처리:**
- 미래 수업 스케줄에서 제거
- 미납 학원비 자동 취소
- 활성 시즌 등록 자동 취소
- 학생 데이터는 삭제되지 않고 "퇴원" 탭에서 확인 가능

### 3.8 졸업 처리하기

고3 또는 N수 학생만 졸업 처리가 가능합니다.

1. 학생 카드에서 **"졸업"** 버튼 클릭
2. 확인 후 졸업 처리

### 3.9 학년 진급

새 학년이 시작될 때 학생들의 학년을 일괄 변경합니다.

1. 학생 목록 화면에서 **"학년 진급"** 버튼 클릭
2. 진급 대상 학생 확인:
   - 고1 → 고2
   - 고2 → 고3
   - 고3 → 졸업 또는 N수
3. **"진급 실행"** 클릭

> **참고:** 매년 3월 1일(신학기)에 자동 진급 스케줄러가 실행됩니다. 원장 권한으로 수동 실행도 가능합니다.

### 3.10 크레딧 시스템 (쉽게 이해하기)

크레딧은 **"나중에 깎아줄 금액"** 입니다.

**쉬운 예시:**
```
홍길동 학생이 이번 달 아파서 3일 공결 처리
→ 3일치 수업료 9만원이 크레딧으로 쌓임
→ 다음 달 학원비 50만원 - 크레딧 9만원 = 41만원만 청구
```

**크레딧이 쌓이는 경우:**
1. **공결** - 사유 있는 결석 (아파서, 학교 행사 등)
2. **휴원** - 휴원 기간 동안 수업 못 받은 만큼
3. **환불 대신** - 현금 환불 안 하고 크레딧으로 전환

**자동으로 처리됩니다:**
- 매월 말일에 시스템이 자동 계산
- 5주차 보너스 수업이 있으면 공결과 상쇄
- 보충 수업 받으면 공결과 상쇄
- 남은 공결만 크레딧으로 변환

**크레딧 확인 방법:**
학생 클릭 → 학원비 탭 → "크레딧 잔액" 확인

**원장이 직접 크레딧 추가하기:**
1. 학생 상세 → 학원비 탭
2. **"크레딧 추가"** 버튼
3. 크레딧 생성 방식 선택:
   - 날짜 범위로 생성 (시작일~종료일)
   - 수업 횟수로 생성 (N회분)
4. 금액과 사유 입력 (예: "시설 문제로 보상")
5. **"추가"** 클릭

**크레딧 관리 탭:**
- 기존 크레딧 목록 확인
- 크레딧 수정/삭제
- 특정 월에 크레딧 적용

## 4. 강사 관리

왼쪽 메뉴에서 **"강사"** 클릭

### 4.1 강사 목록 보기

등록된 모든 강사가 표시됩니다.

**표시되는 정보:**
- 이름
- 연락처
- 강사 유형 (선생님 / 사무보조)
- 급여 유형
- 상태 (재직/퇴직)

**필터:**
- 상태별: 재직 / 퇴직
- 급여 유형별: 시급제 / 건당 / 월급 / 혼합

### 4.2 강사 등록하기

1. **"+ 강사 등록"** 버튼 클릭
2. 기본 정보 입력:

| 항목 | 설명 |
|------|------|
| 이름 | 강사 이름 |
| 연락처 | 전화번호 |
| 성별 | 남/여 |
| 이메일 | 이메일 주소 |
| 주민등록번호 | 급여 신고용 (암호화 저장) |
| 입사일 | 근무 시작일 |

3. 강사 유형 선택:

| 유형 | 설명 |
|------|------|
| **선생님** | 수업을 직접 진행하는 강사 |
| **사무보조** | 학원 운영 보조 인력 |

4. 급여 정보 입력:

| 항목 | 설명 |
|------|------|
| 급여 유형 | 시급제 / 건당 / 월급 / 혼합 |
| 오전 수업 단가 | 오전 수업 1회당 급여 (100원 단위) |
| 오후 수업 단가 | 오후 수업 1회당 급여 (100원 단위) |
| 저녁 수업 단가 | 저녁 수업 1회당 급여 (100원 단위) |
| 세금 유형 | 3.3% / 4대보험 / 비과세 |
| 계좌 정보 | 급여 입금용 계좌 |

**사무보조인 경우 추가 입력:**
- 근무 요일 선택 (월~일)
- 출근 시간 / 퇴근 시간

**세금 유형 설명:**

| 유형 | 설명 | 공제율 |
|------|------|--------|
| **3.3%** | 프리랜서 (사업소득세) | 급여의 3.3% |
| **4대보험** | 정규직/계약직 (근로자) | 약 9.7% |
| **비과세** | 세금 없음 | 0% |

5. **"저장"** 클릭

### 4.3 강사 정보 수정하기

1. 강사 목록에서 수정할 강사 클릭
2. **"수정"** 버튼 클릭
3. 정보 수정 후 **"저장"**

### 4.4 강사 출퇴근 관리

강사의 출퇴근을 기록합니다.

**출근 체크:**
1. 수업 스케줄 화면에서 해당 날짜 선택
2. **"강사 출근"** 버튼 클릭
3. 시간대별(오전/오후/저녁) 강사 출결 체크
4. 출근/퇴근 시간 입력 (선택)
5. **"저장"**

### 4.5 초과 근무 관리

정해진 근무일 외에 추가 근무한 경우:

**초과 근무 요청 (강사/직원):**
1. 수업 스케줄에서 해당 일자 선택
2. **"미배정 출근"** 클릭
3. 요청 날짜, 시간대, 사유 입력
4. **"요청"** 클릭

**초과 근무 승인 (원장):**
1. 수업 스케줄 상단 **"승인 대기"** 버튼에 대기 건수 표시
2. 클릭하면 대기 중인 요청 목록
3. **"승인"** 또는 **"거절"** 클릭

## 5. 수업 스케줄

왼쪽 메뉴에서 **"수업스케줄"** 클릭

### 5.1 스케줄 화면 구성

**캘린더 뷰:**
- 한 달 전체 일정을 달력으로 봅니다
- 각 날짜에 시간대별 강사 수가 표시됩니다 (오전/오후/저녁)
- 강사 아바타 아이콘으로 한눈에 확인
- 상담 일정이 있는 날짜에는 녹색 마커 표시

**리스트 뷰:**
- 테이블 형식으로 수업 목록 확인

### 5.2 날짜별 수업 확인하기

1. 달력에서 원하는 날짜 클릭
2. 해당 날짜의 시간대별 수업이 표시됩니다:

```
┌─────────────────────────────────────────┐
│ 2026년 1월 28일 (수)                     │
├─────────────────────────────────────────┤
│ 오전 (09:00~12:00)                      │
│    담당: 김코치                          │
│    학생: 홍길동, 김철수, 이영희 (3명)     │
│    출석: 2/3 완료                        │
├─────────────────────────────────────────┤
│ 오후 (14:00~17:00)                      │
│    담당: 박코치                          │
│    학생: 박민수, 최지원 외 5명 (7명)      │
│    출석: 미완료                          │
├─────────────────────────────────────────┤
│ 저녁 (18:00~21:00)                      │
│    담당: 이코치                          │
│    학생: 정수현, 강민호 외 3명 (5명)      │
│    출석: 미완료                          │
└─────────────────────────────────────────┘
```

### 5.3 출석 체크하기

1. 날짜 선택 후 시간대 클릭
2. 타임슬롯 상세 모달이 열립니다
3. 학생별로 출석 상태 선택:
   - 출석
   - 결석
   - 지각
   - 공결 (사유 있는 결석 → 크레딧 자동 적립)
4. 결석/공결 시 사유 입력
5. **"저장"** 클릭

**보충 학생 추가:**
- 타임슬롯 상세 모달에서 **"보충 학생 추가"** 버튼
- 다른 시간대 학생을 이번 시간에 추가 수업 가능

### 5.4 학생 시간대 이동하기

학생의 수업 시간대를 변경합니다.

1. 타임슬롯 상세 모달에서 학생 선택
2. 이동할 시간대 버튼 클릭 (오전/오후/저녁)
3. 자동으로 해당 시간대로 이동

### 5.5 강사 배정하기

1. 날짜 선택 후 시간대 클릭
2. 강사 섹션에서 출결 상태 체크
3. 강사별 출근/퇴근 시간 입력 (선택)
4. **"저장"** 클릭

### 5.6 강사 스케줄 패널

수업 스케줄 화면 우측에 강사 스케줄 패널을 열 수 있습니다.

1. 우측의 패널 토글 버튼 클릭
2. 선택한 날짜의 강사별 근무 시간대 표시
3. 강사 배정/변경 가능 (체크박스 선택)

> 패널 열림 상태는 브라우저에 저장되어 다음 접속 시에도 유지됩니다.

### 5.7 수업 자동 생성

매월 1일 00:05에 시스템이 자동으로 수업을 생성합니다:
- 모든 재원 학생의 수업 요일에 맞춰 자동 생성
- 학생의 시간대(오전/오후/저녁) 배정
- 출석 기록 자동 생성

> **수동 생성은 불필요합니다.** 크론 작업이 매월 자동으로 처리합니다.

## 7. 학원비 관리

왼쪽 메뉴에서 **"학원비"** 클릭

### 7.1 학원비 목록 보기

모든 학원비 내역이 표시됩니다.

**표시되는 정보:**
- 학생 이름
- 대상 연월
- 청구 금액
- 납부 금액
- 상태 (미납/부분납부/완납)
- 납부 방법 (계좌이체/카드/현금/기타)

**필터 옵션:**
- 연도/월 선택
- 상태별: 미납 / 부분납부 / 완납
- 결제 방법별: 계좌이체 / 카드 / 현금 / 기타
- 검색: 학생명
- 이전 달 미납자 포함 여부 토글

### 7.2 이번 달 현황

화면 상단에 통계가 표시됩니다:

```
┌───────────────────────────────────────────────┐
│  완납: 42건 (21,000,000원)                     │
│  미납: 8건 (4,000,000원)                       │
│  부분납부: 2건 (500,000원)                     │
└───────────────────────────────────────────────┘
```

### 7.3 학원비 청구 생성하기

**개별 생성:**
1. **"+ 학원비 등록"** 버튼 클릭
2. 학생 선택
3. 대상 연월 선택
4. 청구 금액 확인 (학생 정보에서 자동 입력)
5. 할인 적용 (선택)
6. **"저장"** 클릭

**자동 생성:**
- 매월 1일에 시스템이 자동으로 모든 재원생의 학원비를 생성합니다
- 수동으로 일괄 생성할 필요 없습니다

### 7.4 납부 처리하기

학원비를 납부 받았을 때:

**빠른 납부 처리:**
학원비 목록에서 각 건 우측의 빠른 납부 버튼 사용:
- 계좌 버튼 → 계좌이체 전액 완납 처리
- 카드 버튼 → 카드 전액 완납 처리
- 현금 버튼 → 현금 전액 완납 처리

**상세 납부 처리:**
1. 학원비 목록에서 해당 건 클릭
2. **"납부 처리"** 버튼 클릭
3. 납부 정보 입력:

| 항목 | 설명 |
|------|------|
| 할인 금액 | 이번 건에 한해 할인 적용 |
| 납부 금액 | 실제 받은 금액 |
| 납부 방법 | 계좌이체 / 카드 / 현금 / 기타 |
| 납부일 | 돈을 받은 날짜 |

4. **"저장"** 클릭

**부분 납부:**
- 전액이 아닌 일부만 납부한 경우
- 납부 금액에 실제 받은 금액 입력
- 상태가 "부분납부"로 변경
- 나머지 금액은 "잔액"으로 표시

### 7.5 선납 할인 결제

> v3.8.0에서 추가된 기능입니다.

여러 달치 학원비를 한 번에 납부하면 할인을 받을 수 있습니다.

1. 학생 상세 → 학원비 탭
2. **"선납 결제"** 버튼 클릭
3. 선납 정보 입력:

| 항목 | 설명 |
|------|------|
| 선납 개월수 | 2~6개월 선택 |
| 시작월 | 선납 적용 시작 월 |
| 할인율 | 적용할 할인 비율 |
| 납부 방법 | 계좌이체 / 카드 / 현금 / 기타 |

4. 미리보기에서 월별 금액과 총 납부액 확인
5. **"결제 확정"** 클릭
6. 선택한 개월수만큼 학원비가 한 번에 생성 및 완납 처리

### 7.6 크레딧 차감

학생에게 쌓인 크레딧을 학원비에서 차감합니다.

1. 학원비 상세에서 **"크레딧 적용"** 버튼 클릭
2. 적용할 크레딧 금액 입력
3. **"적용"** 클릭

### 7.7 수동 크레딧 생성

학원비 목록 화면에서 **"크레딧"** 버튼으로 수동 크레딧을 생성할 수 있습니다.

### 7.8 미납자 알림 발송

미납 학원비가 있는 학생/학부모에게 알림톡을 보냅니다.

1. 학원비 목록 상단 **"미납자 알림"** 버튼 클릭
2. 확인 후 전체 미납자에게 알림톡 자동 발송

### 7.9 학원비 수정/삭제

**수정:**
1. 학원비 상세에서 **"수정"** 클릭
2. 금액, 기한 등 수정
3. **"저장"**

**삭제:**
- 이미 납부된 학원비는 삭제할 수 없습니다
- 미납 건만 삭제 가능 (원장 권한 필요)

### 7.10 엑셀 다운로드

학원비 내역을 엑셀 파일로 내보냅니다.

1. **"엑셀 다운로드"** 버튼 클릭
2. 현재 필터 조건에 맞는 데이터가 다운로드됩니다

### 7.11 강사 권한 시 학원비 표시

> v3.6.5에서 추가된 기능입니다.

강사 권한으로 접속 시 학원비 총액이 숨겨지고, 미납 여부만 확인할 수 있습니다.

### 8.1 급여 목록 보기

월별로 강사 급여 현황을 확인합니다.

**화면 구성:**
- 상단: 월 선택 (이전달 | 2026년 1월 | 다음달)
- 목록: 강사별 급여 내역

**표시되는 정보:**
- 강사 이름
- 총 수업 횟수
- 기본급
- 추가 수당
- 공제액
- 실지급액 (10원 단위 이하 버림)
- 지급 상태 (대기/지급완료)

### 8.2 급여 계산하기

1. 월 선택 (예: 2026년 1월)
2. **"급여 계산"** 버튼 클릭
3. 해당 월의 출근 기록을 바탕으로 자동 계산됩니다

**계산 방식 (강사 유형별):**

**선생님 (수업 기반):**
```
기본급 = (오전 수업 횟수 x 오전 단가)
       + (오후 수업 횟수 x 오후 단가)
       + (저녁 수업 횟수 x 저녁 단가)
```

**사무보조 (출근일 기반):**
```
기본급 = 출근 일수 x 시급
```

**혼합형:**
```
기본급 = 월급 + 추가 수당
```

**공통:**
```
세금 = 기본급 x 세율
실지급액 = 기본급 - 세금 + 추가수당 - 공제액
※ 10원 단위 이하 버림
```

### 8.3 급여 상세 확인

1. 강사 이름 클릭
2. 상세 내역 확인:
   - 일별 출근 기록
   - 시간대별 수업 횟수
   - 계산 내역
   - 초과근무 포함 여부

### 8.4 급여 수정하기

자동 계산된 급여를 수정해야 할 때:

1. 강사 급여 상세에서 **"수정"** 클릭
2. 금액 수정:
   - 추가 수당 (보너스 등)
   - 공제액 (선불금 등)
   - 메모
3. **"저장"** 클릭

### 8.5 급여 지급 처리

실제로 급여를 지급했을 때:

**개별 지급:**
1. 강사 급여 상세에서 **"지급 완료"** 클릭
2. 지급일 입력
3. 비밀번호 확인
4. **"확인"**

**일괄 지급:**
1. 급여 목록 상단 **"일괄 지급"** 버튼 클릭
2. 지급할 강사 체크
3. 지급일 입력
4. 비밀번호 확인
5. **"확인"**

### 8.6 급여 재계산

이미 생성된 급여를 출근 기록 변경 후 다시 계산해야 할 때:

1. 강사 급여 상세에서 **"재계산"** 클릭
2. 최신 출근 기록으로 재계산됩니다

### 8.7 급여 내보내기

엑셀 파일로 다운로드합니다.

1. **"엑셀 다운로드"** 버튼 클릭
2. 파일이 자동으로 다운로드됩니다

## 9. 상담 관리

### 9.1 신규 상담

왼쪽 메뉴에서 **"신규상담"** 클릭

학원에 처음 문의하는 학부모/학생의 상담을 관리합니다.

#### 상담 목록 보기

**표시되는 정보:**
- 학생 이름
- 학부모 연락처
- 학년
- 상담 희망일
- 상태 (대기/확정/완료/취소/노쇼)

**필터:**
- 상태별 필터
- 유형별: 문의 / 학습 상담
- 기간별: 전체 / 오늘 / 이번 주

**통계:**
- 각 상태별 건수가 상단에 표시

#### 상담 신청 확인하기

학부모가 공개 페이지에서 신청한 상담이 자동으로 목록에 추가됩니다.

1. 새로운 상담 신청 확인
2. 상담 클릭하여 상세 정보 확인:
   - 학생 정보
   - 성적 정보 (내신, 모의고사)
   - 목표 대학
   - 문의 내용
   - 희망 상담 일시

#### 상담 정보 수정하기

> v3.6.2에서 추가된 기능입니다.

1. 상담 상세에서 **"정보 수정"** 버튼 클릭
2. 학생 정보, 연락처, 성적 등 수정
3. **"저장"** 클릭

#### 상담 확정하기

1. 상담 상세에서 **"확정"** 버튼 클릭
2. 상담 일시 확인/수정
3. **"확정"** 클릭
4. 학부모에게 알림톡이 자동 발송됩니다 (설정된 경우)

#### 상담 진행하기

1. 상담 상세에서 **"상담 진행"** 클릭
2. 상담 내용 기록:
   - 상담 메모
   - 목표 대학 수정
   - 입시 전략 메모
3. **"저장"** 클릭
4. 상담 완료 시 **"완료 처리"** 클릭

#### 체험생으로 전환하기

상담 후 체험 수업을 진행할 때:

1. 상담 상세에서 **"체험 전환"** 클릭
2. 체험 정보 입력:
   - 체험 날짜 선택 (여러 날 가능)
   - 시간대 선택
   - 연락처 확인
3. **"전환"** 클릭
4. 학생 목록에 체험생으로 추가됩니다

#### 미등록 관리로 전환하기

등록을 고민 중인 학부모를 관리할 때:

1. **"미등록 관리 전환"** 클릭
2. 연락처, 메모 입력
3. 학생 목록 → 미등록 탭에서 확인 가능

#### 직접 상담 등록하기

전화나 방문으로 접수된 상담을 직접 등록합니다.

1. **"+ 상담 등록"** 버튼 클릭
2. 상담 유형 선택: 신규 상담 / 재원생 상담
3. 학부모/학생 정보 입력
4. 상담 희망일 선택
5. **"저장"** 클릭

### 9.2 재원생 상담

왼쪽 메뉴에서 **"재원생상담"** 클릭

현재 등록된 학생들의 상담 기록을 관리합니다.

#### 상담 기록 보기

1. 학생 선택
2. 해당 학생의 상담 이력이 표시됩니다

#### 상담 기록 작성하기

1. **"+ 상담 기록"** 버튼 클릭
2. 학생 선택
3. 상담 내용 입력:
   - 상담 유형 (학부모 상담)
   - 상담 날짜
   - 학습 메모
   - 목표 대학
   - 입시 전략
4. **"저장"** 클릭

### 9.3 상담 설정

왼쪽 메뉴 → 신규상담 → 우측 상단 **"설정"**

#### 상담 운영 시간 설정

1. 요일별 상담 가능 여부 설정 (월~일)
   - 운영 여부 체크박스
   - 시작 시간
   - 종료 시간
2. **"저장"** 클릭

#### 상담 불가 시간 설정

특정 시간에 상담을 받지 않을 때:

1. **"차단 시간 추가"** 클릭
2. 날짜와 시간 선택
3. **"추가"** 클릭

### 9.4 상담 캘린더

왼쪽 메뉴 → 신규상담 → **"캘린더"** 탭

달력 형태로 상담 일정을 확인합니다.

- 날짜별 상담 수 표시
- 클릭하면 해당 날짜 상담 목록
- 색상으로 상태 구분:
  - 노란색: 대기
  - 초록색: 확정
  - 파란색: 완료
  - 회색: 취소

> **수업 스케줄에서 연동:** 수업 스케줄 캘린더에서 상담 마커를 클릭하면 해당 날짜의 상담 캘린더로 이동합니다.

## 10. 시즌 관리

왼쪽 메뉴에서 **"시즌"** 클릭

수시/정시 시즌을 관리합니다.

### 10.1 시즌이란?

입시 시즌(수시/정시)에 맞춰 학생들을 그룹으로 관리하고, 시즌비를 청구하는 기능입니다.

**예시:**
- 2026 수시 시즌 (7월~9월)
- 2026 정시 시즌 (11월~1월)

### 10.2 시즌 목록 보기

**표시되는 정보:**
- 시즌명
- 시즌 유형 (수시/정시)
- 기간
- 기본 시즌비
- 등록 학생 수
- 상태 (예정/진행중/종료)

### 10.3 시즌 만들기

1. **"+ 시즌 만들기"** 버튼 클릭
2. 기본 정보 입력:

| 항목 | 설명 |
|------|------|
| 시즌명 | 예: "2026 수시 시즌" |
| 시즌 유형 | 수시 / 정시 |
| 시작일 | 시즌 시작 날짜 |
| 종료일 | 시즌 종료 날짜 |
| 기본 시즌비 | 시즌 등록 기본 금액 |
| 설명 | 시즌 안내 (선택) |

3. **"저장"** 클릭

### 10.4 시즌에 학생 등록하기

1. 시즌 목록에서 시즌 클릭
2. **"학생 등록"** 버튼 클릭
3. 등록할 학생 선택 (여러 명 가능, 체크박스)
4. 학생별 시즌비 확인/수정
5. 등록 전 미리보기 확인
6. **"등록"** 클릭

**일할 계산:**
시즌 중간에 등록하는 경우 자동으로 일할 계산됩니다.

```
예시: 30일 시즌, 20일차에 등록
시즌비 = 기본 시즌비 x (남은 일수 / 전체 일수)
       = 300,000원 x (10일 / 30일) = 100,000원
```

### 10.5 시즌 등록 학생 관리

1. 시즌 상세에서 등록 학생 목록 확인
2. 학생별 정보:
   - 시즌비
   - 납부 상태
   - 등록일

### 10.6 시즌비 납부 처리

1. 시즌 상세 → 학생 클릭
2. **"납부 처리"** 버튼 클릭
3. 납부 정보 입력
4. **"저장"**

### 10.7 시즌 등록 취소/환불

학생이 시즌 중간에 그만두는 경우:

1. 시즌 상세 → 학생 클릭
2. **"등록 취소"** 버튼 클릭
3. 취소일 선택
4. 환불 금액 확인 (자동 계산):

**환불 계산 (학원법 기준):**
```
수강 기간에 따른 환불률:
- 1/3 경과 전: 수강료의 2/3 환불
- 1/2 경과 전: 수강료의 1/2 환불
- 1/2 경과 후: 환불 없음
```

5. 환불 방법 선택:
   - 현금 환불
   - 크레딧 전환
6. 부가세 옵션 선택 (선택)
7. **"취소 처리"** 클릭

**환불 명세서:**
- 취소 시 환불 명세서가 생성됩니다
- 복사, 인쇄 가능
- 환불 기록이 지출 관리에 자동 등록됩니다

## 11. 문자 보내기

왼쪽 메뉴에서 **"문자 보내기"** 클릭

학생/학부모에게 문자를 보냅니다.

### 11.1 문자 보내기 화면

**구성:**
- 왼쪽: 받는 사람 선택
- 오른쪽: 문자 내용 작성

### 11.2 받는 사람 선택하기

**전체 선택:**
- "전체 학생" - 모든 재원생에게 발송
- "전체 학부모" - 모든 학부모에게 발송

**조건 선택:**
- 상태별: 재원생 / 휴원 / 체험 등
- 학년별: 고1 / 고2 / 고3 / N수

**직접 입력:**
- **"직접 입력"** 탭 선택
- 전화번호 직접 입력 (한 줄에 하나씩)

**학생 전화 vs 학부모 전화:**
- 각 학생별로 학생 본인 번호 또는 학부모 번호 선택 가능

### 11.3 문자 작성하기

1. 문자 내용 입력
2. 문자 종류 자동 선택:
   - SMS: 80바이트(한글 약 40자) 이하
   - LMS: 80바이트 초과
   - MMS: 이미지 첨부 시

**이미지 첨부:**
1. **"이미지 추가"** 버튼 클릭
2. 이미지 파일 선택
3. MMS로 자동 전환

### 11.4 발송하기

1. 받는 사람 수 확인
2. 예상 요금 확인
3. **"발송"** 버튼 클릭
4. 확인 후 발송

### 11.5 발송 내역 확인

**"발송 내역"** 탭에서 확인:
- 발송 일시
- 받는 사람 수
- 성공/실패 건수
- 문자 내용

### 11.6 발신번호 관리

문자 발송 시 표시되는 전화번호를 설정합니다.

1. **설정** 페이지 → 알림톡 설정에서 관리
2. 발신 번호 등록 (사업자 등록된 번호)

> 발신번호는 실제 소유한 번호만 등록 가능합니다.

## 19. 상담 신청 페이지 (공개 페이지)

학생 또는 학부모 누구나 직접 상담을 신청할 수 있는 공개 페이지입니다.
로그인이 필요 없습니다.

### 19.1 페이지 주소

`paca.vercel.app/c/학원주소`

예시: `paca.vercel.app/c/서울체대입시`

### 19.2 신청 과정

**1단계: 학생 정보 입력**
- 학생 이름
- 연락처
- 학년 (중3~성인)
- 학교
- 성별

**2단계: 성적 및 입시 정보 입력**
- 내신 평균등급 (1~9등급)
- 모의고사 성적 (국어, 수학, 영어, 탐구)
- 입시 전형 (수시 / 정시 / 수시+정시)
- 목표 대학
- 소개해주신 재원생

**3단계: 상담 일정 선택**
- 달력에서 날짜 선택 (학원 운영시간만 표시)
- 가능한 시간 중 선택
- 문의 내용 작성

**4단계: 신청 완료**
- 예약 번호 발급
- 확인 문자 발송 (설정 시)

### 19.3 예약 변경

학부모가 예약 번호로 상담 일정을 변경할 수 있습니다.

`paca.vercel.app/consultation/예약번호`

1. 예약 번호로 접속 (로그인 불필요)
2. 새 날짜/시간 선택
3. **"변경"** 버튼 클릭

### 19.4 학원에서 확인하기

상담 신청이 들어오면:
1. 신규상담 목록에 자동 추가
2. 푸시 알림 발송 (설정된 경우)
3. 원장이 확인 후 **"확정"** 처리
4. 신청자에게 확정 알림톡 발송

## 20. 자동화 기능

P-ACA는 다양한 작업을 자동으로 처리합니다.

### 20.1 월간 자동 처리 (매월 1일)

| 시간 | 작업 | 설명 |
|------|------|------|
| 00:05 | 수업 자동 생성 | 모든 재원생의 수업 스케줄 자동 생성 |
| 이후 | 학원비 자동 생성 | 재원생 학원비 청구 자동 생성 |

### 20.2 매일 자동 처리

| 작업 | 설명 |
|------|------|
| 체험 만료 체크 | 체험 기간 종료 학생 상태 업데이트 |
| 휴원 종료 체크 | 휴원 종료일 도래 학생 알림 |
| 공결 크레딧 처리 | 공결 처리된 출석 → 크레딧 자동 생성 |

### 20.3 실시간 자동 처리

| 작업 | 설명 |
|------|------|
| 상담 리마인드 | 상담 30분 전 자동 알림 |
| 미납 출석 알림 | 미납 학생이 출석하면 푸시 알림 (매일 18시, 21시) |

### 20.4 연간 자동 처리

| 시기 | 작업 | 설명 |
|------|------|------|
| 3월 1일 | 학년 진급 | 중1→중2, 중2→중3, 중3→고1, 고1→고2, 고2→고3, 고3→N수 자동 진급 |

## 21. PWA 앱 설치 및 관리

P-ACA는 PWA(Progressive Web App)로 앱스토어 없이 설치할 수 있습니다.

### 21.1 설치 방법

브라우저에서 접속 시 **"홈 화면에 추가"** 또는 **"앱 설치"** 알림이 자동으로 표시됩니다.

**알림이 안 보이는 경우:**
- 이미 설치한 경우 표시되지 않습니다
- 24시간 이내에 닫은 경우 다시 표시되지 않습니다
- 브라우저 설정에서 직접 설치:
  - 크롬: 주소창 오른쪽 설치 아이콘
  - 사파리: 공유 → 홈 화면에 추가

### 21.2 자동 업데이트

앱이 업데이트되면:
1. 앱 실행 시 자동으로 새 버전 감지
2. 캐시를 자동으로 클리어
3. 새 버전으로 자동 새로고침
4. 로그인 상태는 유지됩니다

### 21.3 화면이 업데이트되지 않을 때

간혹 이전 버전이 계속 표시되는 경우:

**방법 1: 강제 새로고침**
- PC: `Ctrl + Shift + R` (Windows) / `Cmd + Shift + R` (Mac)
- 모바일: 앱 완전 종료 후 재실행

**방법 2: 앱 재설치**
1. 기존 앱 삭제
2. 브라우저로 다시 접속
3. 앱 재설치

## 22. 자주 묻는 질문

### 로그인/계정

**Q: 비밀번호를 잊어버렸어요**
A: 로그인 화면 → "비밀번호 찾기" → 이메일 입력 → 이메일로 받은 링크 클릭 → 새 비밀번호 설정

**Q: 직원 계정은 어떻게 만드나요?**
A: 관리 → 직원관리 → "+ 권한 부여" (원장만 가능)

**Q: 태블릿에서 로그인이 안 돼요**
A:
1. 인터넷 연결 확인
2. 주소가 `paca.vercel.app/tablet`인지 확인
3. 이메일, 비밀번호 정확히 입력
4. 문제 지속 시 앱 삭제 후 재설치

---

### 학생 관리

**Q: 학생을 완전히 삭제하고 싶어요**
A: 학생 데이터는 완전 삭제되지 않고 "퇴원" 처리됩니다. 이는 기록 보관을 위한 것입니다. 퇴원 처리하면 목록에서 보이지 않습니다 (퇴원 탭에서만 확인 가능).

**Q: 체험생이 정식 등록하면 어떻게 하나요?**
A: 학생 상세 → "정식 등록" 버튼 클릭 → 수업 정보 입력 → 저장

**Q: 휴원 중인 학생의 학원비는 어떻게 되나요?**
A: 휴원 처리 시 학원비가 자동으로 일할 계산 조정됩니다. 환불 선택 시 환불 기록도 자동 생성됩니다.

**Q: 학생 검색이 안 돼요**
A: 이름이나 전화번호로만 검색됩니다. 보안상 암호화된 필드이므로 정확한 이름 또는 전화번호를 입력해보세요.

---

### 수업/출석

**Q: 수업이 자동으로 생성되지 않아요**
A:
1. 학생의 수업 요일이 설정되어 있는지 확인
2. 학생 상태가 "재원"인지 확인
3. 매월 1일 00:05에 자동 생성됨 (수동 생성 불필요)

**Q: 출석 체크를 잘못했어요**
A: 해당 날짜 수업의 타임슬롯 클릭 → 학생의 출석 상태 수정 → 저장

**Q: 공결 처리하면 뭐가 달라지나요?**
A: 공결 처리하면 해당 학생에게 크레딧이 자동 적립됩니다. 이 크레딧은 학원비에서 차감할 수 있습니다.

---

### 학원비/결제

**Q: 학원비는 언제 자동으로 생성되나요?**
A: 매월 1일에 자동으로 모든 재원생의 학원비가 생성됩니다.

**Q: 할인은 어떻게 적용하나요?**
A:
- 학생별 고정 할인: 학생 등록/수정에서 할인율/금액 설정
- 개별 할인: 납부 처리 시 할인 금액 입력
- 선납 할인: 학원비 → 선납 결제로 여러 달 한 번에 납부

**Q: 학원비를 잘못 입력했어요**
A: 납부 전이면 수정/삭제 가능합니다. 납부 후에는 수정만 가능합니다.

---

### 상담

**Q: 상담 신청 페이지가 안 열려요**
A:
1. 설정 → 상담 페이지 설정에서 주소 확인
2. 상담 운영 시간이 설정되어 있는지 확인

**Q: 상담 알림톡이 안 가요**
A:
1. 설정 → 알림톡 설정에서 발신번호가 등록되어 있는지 확인
2. 자동 발송이 켜져 있는지 확인
3. "테스트 발송"으로 확인

---

### 시즌

**Q: 시즌 중간에 등록하면 시즌비는 어떻게 되나요?**
A: 남은 기간만큼 일할 계산되어 청구됩니다. 등록 화면에서 자동 계산된 금액을 확인할 수 있습니다.

**Q: 시즌 환불 금액은 어떻게 계산되나요?**
A: 학원법 기준으로 자동 계산됩니다:
- 1/3 경과 전: 2/3 환불
- 1/2 경과 전: 1/2 환불
- 1/2 경과 후: 환불 없음

---

### 기타

**Q: 데이터 백업은 어떻게 하나요?**
A: 데이터는 서버에 자동으로 저장됩니다. 리포트 → 엑셀 다운로드로 별도 보관할 수 있습니다.

**Q: 화면이 이상하게 보여요**
A:
1. 강제 새로고침: `Ctrl + Shift + R` (PC) / 앱 재실행 (모바일)
2. 앱 삭제 후 재설치
3. 다른 브라우저로 접속

**Q: 문의는 어디로 하나요?**
A: 전화 010-2144-6755

### 직원 권한 상세

직원 계정은 각 기능별로 **보기**와 **수정** 권한을 별도로 설정할 수 있습니다.

- **보기(view)**: 해당 페이지에 접근하여 데이터를 볼 수 있음
- **수정(edit)**: 데이터를 추가, 수정, 삭제할 수 있음

예시:
- 학원비 "보기"만 있는 직원 → 미납 현황만 확인 가능
- 학생 "보기+수정" 있는 직원 → 학생 정보 조회 및 수정 가능
- 급여 권한 없는 직원 → 급여 메뉴 자체가 보이지 않음
