Recommendation

1차 추천: Direction A (Operations Desk) — 운영자 기본값

P-ACA는 PC 기반 운영 도구이고, 가장 자주 보는 화면이 '학생 명단 + 출결/결제/상담 빠른 조회'다. A는 그 작업에 가장 직접적인 밀도와 위계를 제공하면서, 기존 사용자의 어드민 근육을 굳히지 않는다. 한국 운영자에게 익숙한 '왼쪽 명단 + 오른쪽 상세' 패턴의 현대판이라고 보면 된다.

A를 추천하는 이유
  • 운영 시간의 대부분이 테이블 + 검색 위주 — A가 가장 빨리 도달
  • 142명 학생의 출석/결제 상태를 한눈에 — KPI 6개가 압축 요약
  • ⌘K 커맨드바는 강사/상담 전환의 마찰을 0으로 만듦
  • 한국 데스크 운영자에게 가장 낮은 학습비용
  • 백엔드 라우팅/테이블 컴포넌트 변경 최소 — 출결·결제 빈 데이터 그대로 활용
A의 약점과 보완
  • '오늘을 훑는' 매니저 뷰는 약함 → B의 3-panel 스캔 KPI를 A 헤더 아래 보조 row로 차용
  • 태블릿에서 큐 사라짐 → A 큐를 하단 sheet로 살리는 v2 작업
  • 현장 동선(체육관/운동장)은 약함 → 운동장 진입 시에만 C의 큰 액션 모드 토글 (별도 라우트)

각 방향은 prototypes/ 아래 단일 HTML 파일로 분리되어 있어, 브라우저에서 직접 열거나 design-demos/와 결합해 비교 가능합니다.
